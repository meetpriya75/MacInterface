# 🍎 MacOS Desktop Simulator

A fully interactive **MacOS-inspired desktop environment** built with React, featuring draggable windows, a functional CLI, and app-like navigation — designed to replicate a real desktop experience inside the browser.

---

## 🚀 Features

* 🖥️ **MacOS-like UI** with desktop layout
* 🪟 **Draggable & Resizable Windows**
* 💻 **Interactive CLI Terminal** (custom commands)
* 📂 **Multiple Apps Simulation**

  * Notes
  * GitHub
  * Calendar
  * PDF Viewer
  * Spotify Embed
  * Messaging UI
* ⚡ Smooth animations and responsive interactions

---

## 🛠️ Tech Stack

* **Frontend:** React, JavaScript
* **Styling:** SCSS
* **Build Tool:** Vite
* **Version Control:** Git & GitHub

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Shraysinha/macos-desktop-simulator.git

# Navigate into the project
cd macos-desktop-simulator

# Install dependencies
npm install

# Run the development server
npm run dev
```
